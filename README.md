# Career-Path-Recommendation
Machine Learning based Career Path Recommender System built with Python, Flask, Scikit-Learn, and SQLite that recommends suitable careers based on skills, education, certifications, and academic performance.
Features
- Login & Signup
- Career Prediction
- Salary Insights
- Skill Gap Analysis
- Learning Roadmap
- Prediction History
- PDF Report Download

Tech Stack
- Python
- Flask
- Pandas
- Scikit-Learn
- SQLite
- HTML/CSS

ML Model
- Random Forest Classifier
